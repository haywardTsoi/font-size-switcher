# Demo Instructions

To create a demo GIF for the Font Size Switcher plugin, follow these steps:

## Recording the Demo

1. Set up a Filament application with the Font Size Switcher plugin
2. Open the admin panel in your browser
3. Use a screen recording tool (like QuickTime on macOS or OBS Studio)
4. Record the following sequence:

### Demo Sequence

1. **Show the default state** - Display the admin panel with the font size switcher visible in the top bar
2. **Click small size button** - Show the interface text becoming smaller
3. **Click large size button** - Show the interface text becoming larger  
4. **Click medium size button** - Show returning to normal size
5. **Navigate between pages** - Show that the font size persists across pages
6. **Toggle dark mode** (if available) - Show that the switcher works in dark mode

### Tips for Good Demo

- Record at 1920x1080 resolution
- Keep the recording under 30 seconds
- Show clear contrast between font sizes
- Include some text-heavy pages (like forms or tables)
- Make sure the font size switcher buttons are clearly visible
- Show the hover effects on the buttons

### Converting to GIF

Use a tool like:
- [GIPHY Capture](https://giphy.com/apps/giphycapture) (macOS)
- [ScreenToGif](https://www.screentogif.com/) (Windows)
- [Peek](https://github.com/phw/peek) (Linux)

Or use online converters like:
- [CloudConvert](https://cloudconvert.com/mp4-to-gif)
- [EZGIF](https://ezgif.com/video-to-gif)

### Final Steps

1. Save the GIF as `demo.gif` in the root directory
2. Update the README.md file to reference the correct path
3. Optimize the GIF size (should be under 5MB for GitHub)

## Example Demo Script

"Here's the Font Size Switcher plugin in action:
1. The switcher appears in the top-right corner of Filament
2. Click the small 'A' for smaller text
